## 🌟 Overview

**SHIELD-DESIGN** is a cutting-edge, agent-agnostic HTML presentation and design engine. Engineered with modern web technologies, it allows you to separate content from presentation flawlessly. Define your slides in a clean `manifest.json`, generate high-fidelity interactives, and watch as the engine renders a stunning, fully animated, responsive presentation within minutes.

---

## ✨ Key Features

- 🚀 **Manifest-Driven Content** - Keep data and styling separated. Swap `manifest.json` without touching component code.
- 🎨 **Premium Aesthetic Themes** - High-fidelity, elegant, and fully customizable visual identity preventing AI slop.
- 🧩 **Dynamic Slide Renderer** - Driven by flexible `type`, `layoutVariant`, and `theme` combinations.
- 📱 **Interactive App Prototypes** - Single-file HTML iOS mockups with native bezels and automated Playwright validation.
- 🖨️ **Automated Export Pipelines** - Built-in Playwright scripts for seamless 60fps `MP4`, `GIF`, Editable `PPTX`, and `PDF` generation.
- ✅ **Type-Safe Validation** - Strong runtime checks using `Zod` over `content/manifest.json`.

## 🚀 Getting Started

### Installation
Clone the repository and install the required dependencies:
```bash
# Install dependencies
npm install
```

### Running the App
```bash
# Start the development server
npm run dev

# Build for production
npm run build
```

---

## 📸 Export Automation

SHIELD-DESIGN includes automated pipelines to generate static assets and videos from your presentations.

```bash
# Export the entire deck as a high-quality PDF
npm run export:pdf

# Export the presentation directly into an editable PPTX file
npm run export:pptx

# Render a buttery-smooth 60fps MP4 with dual-track audio (BGM + SFX)
npm run export:video
```
